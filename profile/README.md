<details>
<summary align="center"><b>ЁЯдЭ How to Contribute</b></summary>
<br>
<div align="center">

Got a printer profile, filament profile, Klipper macro, or firmware config worth sharing? We want it.

**1.** Fork any repo in this org
**2.** Add your profiles/configs in the appropriate folder
**3.** Document what printer, filament, and slicer version it targets
**4.** Open a Pull Request тАФ it'll be reviewed fast

Have a feature idea, bug report, or just want to say hi?
